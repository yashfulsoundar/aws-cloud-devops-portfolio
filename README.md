# ☁️ Cloud & DevOps Portfolio

A modern and responsive personal portfolio website built to showcase my skills, projects, and learning journey in Cloud Computing and DevOps.

## 🚀 Features

- Responsive Design
- Modern UI
- About Me Section
- Technical Skills
- Projects Section
- Experience Section
- Contact Form
- Thank You Page
- Resume Download
- GitHub Integration
- Smooth Navigation

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- FormSubmit (Contact Form)
